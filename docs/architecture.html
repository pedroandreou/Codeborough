Privacy boundaryHow it actually runs — four Docker containers on two networks. The reasoning core (vLLM + gateway) sits on core_net (internal: true), so it has no internet route at all. Exactly one container, the default-deny egress-proxy, may reach out — and only to ElevenLabs. The single inbound is the UI on :8091. Prove it live with make prove-boundary.
Voice turn 1 you speak (Scribe STT) 2 OpenClaw Gateway 3 Nemotron reasons 4 civic-geo tool call 5 local London GeoJSON lookup 6 the grounded answer returns through OpenClaw to Eleven v3 7 it is spoken back, you hear. Session memory saves and recalls every turn, a side loop rather than a numbered step.
Brain, data and memory are fully on-device; only the voice in and out goes through ElevenLabs, which that bounty requires.
